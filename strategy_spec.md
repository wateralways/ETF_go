# 大盘急跌-非过热板块超跌反弹策略

## 策略概述

当大盘（沪深300）出现恐慌性单日大跌时，买入前期未被资金追捧（非过热）的科技/制造类 ETF，博取恐慌后 1~2 周的反弹收益。

**核心逻辑**：大盘急跌时，前期冷门板块因筹码结构干净、无获利盘堆积，跟跌后反弹弹性更大。前期热门板块反而面临解套抛压。

## 交易规则（完整流程）

### 每日收盘后：

**Step 1 — 判断大盘是否急跌**
```
沪深300 当日涨跌幅 ≤ -0.7% → 触发，进入 Step 2
否则 → 不操作
```

**Step 2 — 计算每只 ETF 的"过热度"**
```
过热度 = ETF 过去 10 个交易日的累计涨跌幅

累计涨跌幅 = (∏(1 + 每日涨跌幅/100) - 1) × 100
```

**Step 3 — 筛选非过热 ETF**
```
1. 所有 ETF 按过热度从小到大排序（越冷排越前）
2. 取前 50%（即过去 10 天涨得最少的那一半）
3. 从中剔除急跌日跌幅 > -0.3% 的 ETF（即跌太少的不买，确保是恐慌跟跌）
```

**Step 4 — 买入**
```
急跌日收盘价，等权买入所有符合条件的 ETF
```

**Step 5 — 卖出**
```
分别按 T+2、T+5、T+8、T+10 收盘价卖出，统计各持有期收益
（实际交易中择一最优持有期即可）
```

## 优化方向（待探索）

- **过热度窗口**：对比 5/10/15/20 日窗口的表现
- **非过热比例**：对比后 30%/50%/70% 的表现
- **急跌阈值**：对比 -0.5%/-0.7%/-1.0%/-1.5% 的触发频率和收益
- **增加止损**：设置 -5%/-8% 止损对收益曲线的影响
- **大盘方向过滤**：仅在上升趋势中执行（如沪深300在 MA60 上方）
- **候选池扩展**：纳入消费、医药、周期类 ETF，扩大候选范围

---

## 风险提示

- 本策略基于历史数据回测，过去表现不代表未来收益
- 极端系统性风险（如 2025-04-07 沪深300 跌 -7.05%关税冲击）下策略仍有效，但单笔回撤可能很大
- 候选 ETF 可能存在流动性风险，实盘需关注日均成交额
- 建议结合大盘趋势过滤，在熊市中降低仓位或暂停策略
