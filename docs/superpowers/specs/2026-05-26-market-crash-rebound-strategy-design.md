# 大盘急跌-非过热板块超跌反弹策略

## 策略概要

当沪深300单日急跌时，买入前期非过热的科技/制造类ETF，博取恐慌后反弹收益。

## 触发条件

- **大盘急跌**：沪深300 单日涨跌幅 ≤ -0.7%
- **非过热判定**：过去10个交易日累计涨跌幅排名后50%
- **跟跌确认**：急跌当日 ETF 涨跌幅 ≤ -0.3%

## 交易规则

- **买入**：急跌日收盘，等权买入所有符合条件的ETF，满仓
- **卖出**：分别统计 T+2、T+5、T+8、T+10 收盘卖出的收益
- **无止损/止盈**

## 回测参数

- **数据源**：Tushare Pro
- **回测区间**：2025-01-01 至今；单独分析 2026-03 ~ 2026-05
- **基准指数**：沪深300（000300.SH）

## 代码结构

单文件 `backtest.py`：
1. 数据获取 — Tushare拉取沪深300和ETF日线
2. 信号生成 — 标记急跌日，计算过热度，筛选买入标的
3. 模拟交易 — 按信号买入，按持有期分别统计卖出收益
4. 结果输出 — 胜率、平均收益、累计收益曲线、分持有期对比

## 输出指标

- 每个持有期的胜率、平均收益率、最大单笔收益、最大单笔亏损、交易次数
- 累计收益曲线（4条持有期对比）
- 2025全年 + 2026年3-5月两份独立报告
